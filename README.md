# react-native-smart-qr-scanner

A smart QR scanner view

## Installation


```sh
npm install react-native-smart-qr-scanner
```


## Usage


```js
import { SmartQrScannerView } from "react-native-smart-qr-scanner";

// ...

<SmartQrScannerView color="tomato" />
```
